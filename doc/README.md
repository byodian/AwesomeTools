### TextExpander 的进阶功能有这些

- 插入预设文本：通过快捷短语来输入文本。
- 插入日期时间：包括日期时间的加减。
- 代入其他片段：可以引用已有的其他内容。
- 代按特殊按键：比如输入邮箱后自动按下 Tab。
- 模板填空功能：比如制作一份可调整的模板。
- 输入缩写后改变光标位置：比如在输入 \*\* 之后移动光标到两者之间。
- 剪贴板和图片调用：比如输入剪贴板中的内容、插入图片。
- 快捷运行脚本：比如运行 JS 等代码。
